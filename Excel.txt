1. Formatting

What is Formatting?

Formatting means changing the appearance of your data to make it neat and easy to read.

Example

Before Formatting

name	age	department
varsha	20	it

After Formatting

Name	Age	Department
Varsha	20	IT
What you can do
Change font
Increase font size
Make text Bold
Italic
Underline
Change text color
Fill cell color
Add borders
Center or align text
Format numbers, dates, currency, percentages
Example
Before	After
1000	₹1,000
0.75	75%
12/5/25	12-May-2025
2. Tables
What is a Table?

A table is a structured range of data with headers.

Normal Data
Name	Age	Department
Varsha	20	IT
Priya	19	CSE
Convert to Table

Select data

Press

Ctrl + T

or

Insert → Table

Excel automatically adds:

Filter buttons
Alternate row colors
Easy sorting
Structured references
Advantages

✅ Easy filtering

✅ Easy sorting

✅ Auto formatting

✅ Automatically expands when new rows are added

4. Filtering
What is Filtering?

Filtering displays only the data you want while hiding the rest.

Example
Name	Department
Varsha	IT
Priya	CSE
Rahul	IT

If you filter

Department = IT

Output

Name	Department
Varsha	IT
Rahul	IT

Priya is hidden but not deleted.

5. Freeze Panes
What is Freeze Panes?

Freeze Panes keeps important rows or columns visible while you scroll.

Suppose your data has 10,000 rows.

When you scroll down,

you can't see

| Name | Age | Department |

Freeze Panes keeps these headers visible.

Excel Steps

View

↓

Freeze Panes

↓

Freeze Top Row

Now the first row remains visible.

You can also freeze

First Column
Both Rows and Columns

What is Conditional Formatting?

Conditional Formatting changes the color or style of cells based on rules.

Example

Marks

Suppose we say

Make Green
