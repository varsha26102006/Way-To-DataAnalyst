Phase 4 DAX (Week 3)

DAX is the formula language used in Power BI.

SUM

Adds values.

Total Sales = SUM(Sales[Sales])
COUNT

Counts rows.

Orders = COUNT(Sales[OrderID])
DISTINCTCOUNT

Counts unique values.

Customers = DISTINCTCOUNT(Sales[CustomerID])
CALCULATE ⭐⭐⭐⭐⭐

Changes the filter context.

Example

South Sales =
CALCULATE(
SUM(Sales[Sales]),
Sales[Region]=South
)
FILTER

Filters rows.

FILTER(
Sales,
Sales[Sales]500
)
IF
Profit Status =
IF(
Sales[Profit]0,
Profit,
Loss
)
SWITCH

Works like multiple IF statements.

Grade =
SWITCH(
TRUE(),
Sales[Sales]1000,High,
Sales[Sales]500,Medium,
Low
)
RANKX

Ranks values.

Rank =
RANKX(
ALL(Sales),
SUM(Sales[Sales])
)

Example

Product	Sales	Rank
A	900	1
B	800	2
C	500	3

Phase 5 Visualizations (Week 4)

Learn when to use each visual.

Visualization	Purpose
Card	Show a single KPI like Total Sales or Profit
KPI	Compare current value with a target or previous period
Line Chart	Show trends over time (Sales by Month)
Bar Chart	Compare categories (Sales by Product, Region, Customer)
Matrix	Interactive pivot table with drill-down
Table	Display detailed records and values
Map	Show geographic data such as Sales by State or City
Treemap	Compare category sizes using nested rectangles
Donut Chart	Show percentage contribution of categories
Slicers	Add interactive filters (Date, Region, Category, etc.)
